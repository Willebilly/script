# Applied script, 20 yhp @Frans Schartau

Detta repository innehåller övningar för kursen Applied Script. Som genomförder hösten 2025. Övningarna demonstrerar praktisk tillämning av scripting, versionshantering och systemadministration i Linux-miljöer.

Mål
Målet med kursen är att ge en grundläggande förståelse för automatisering, Bash och scripting, samt hur scripting kan användas för att förenkla uppgifter och effektivisera arbetsflöden.

Projektet som gjordes för kursen ligger i repository Automated-Networkscanner
