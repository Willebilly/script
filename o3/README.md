Övning 3

Detta script tittar på vilket operativ system som enheten använder. Om 
den använder WINDOWS kommer programet att skapa en ny fil med en känd 
Virus signatur från EICAR. Den är i sig inte skadlig men kan användas för
att se hur Antivirus jobbar och fungerar för att ta bort filen.

Slutligen får man också en notis på att filen tagits bort då det går
snabbt för WINDOWS att upptäcka och ta bort den. :D
